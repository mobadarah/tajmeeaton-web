# PyQuran

التقنيات: Data Analysis, Python
وسوم: القرآن الكريم, اللغة عربية
نوع المبادرة: مكتبة برمجية
الوصف بالعربية: هي حزمة بايثون توفر أدوات لتحليل القرآن الكريم والنصوص العربية. تتيح للمستخدمين استرجاع نصوص القرآن، البحث المتقدم، التحويل بين أنظمة الأبجدية المختلفة، وإنشاء جداول تكرار الكلمات.
الوصف بالإنجليزية:  Python package that provides tools for Quranic analysis and Arabic texts. It allows users to retrieve Quranic text, perform advanced searches, transliterate between different alphabetical systems, and generate word frequency tables.
