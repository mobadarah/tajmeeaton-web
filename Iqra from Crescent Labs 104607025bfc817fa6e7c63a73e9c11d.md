# Iqra from Crescent Labs

التقنيات: CSS, Flask, HTML, Java, JavaScript, ReactJS, Swift
وسوم: أصوات, القرآن الكريم
نوع المبادرة: RESTful API, تطبيق iOS, تطبيق أندرويد
الوصف بالعربية: إقرأ هو أداة تهدف إلى تمكين المسلمين من البحث في القرآن باستخدام التعرف على الصوت. تتوفر أيضًا مستودعات للموقع الإلكتروني، وتطبيق الأندرويد، وتطبيق نظام iOS. يعتمد الموقع في خلفيته على خادم مبني بإطار "فلاسْك"، بينما تُبنى الواجهة الأمامية باستخدام مكتبة "ريآكت" مع "ريدوكس". وتُطوَّر تطبيقات الهاتف المحمول باستخدام تقنيات برمجية أصلية لكل نظام مثل الجافا للأندرويد.
الوصف بالإنجليزية: Iqra is a tool meant to allow Muslims to search the Quran using speech recognition. There are also repos for the website, the Android client, and iOS client. The website's back-end is a Flask server, and the front-end is React combined with Redux. Mobile apps are built using native technologies.
