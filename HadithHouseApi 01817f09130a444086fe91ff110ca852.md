# HadithHouseApi

التقنيات: Angular, Backend, Django, Elasticsearch, Grunt, Jenkins, PostgreSQL, Python, TypeScript
وسوم: إسلام, الحديث النبوي
نوع المبادرة: RESTful API
الوصف بالعربية: يُعد مشروع HadithHouseApi بمثابة الواجهة الخلفية لمشروع Hadith House، الذي يهدف إلى جمع وتصنيف الأحاديث النبوية الشريفة. يوفر هذا المشروع واجهات برمجة تطبيقات (APIs) تُمكّن المطورين من الوصول إلى قاعدة بيانات الأحاديث، بما في ذلك تفاصيل السند والمتن، مع إمكانية البحث والتصفية.
الوصف بالإنجليزية: HadithHouseApi serves as the backend for the Hadith House project, which aims to collect and classify the sayings of Prophet Muhammad (Hadith). This project provides APIs that allow developers to access the Hadith database, including details of the chain of narration (Isnad) and the text (Matn), with capabilities for search and filtering.
