# Tafqit

التقنيات: JavaScript
وسوم: اللغة عربية, برمجة, مكتبة, ويب
نوع المبادرة: مكتبة برمجية
الوصف بالعربية: مكتبة “تفقيط” هي مكتبة جافاسكريبت مفتوحة المصدر تهدف إلى تحويل الأرقام إلى نصوص باللغة العربية باستخدام نظام "تفقيط". يمكن للمكتبة التعامل مع الأرقام الصحيحة والأرقام العشرية، وتحويلها إلى كلمات عربية مثل تحويل الرقم "123" إلى "مئة وثلاثة وعشرون". تُعد هذه المكتبة مفيدة في تطبيقات مثل الفواتير، المستندات المالية، أو أي تطبيق يتطلب كتابة الأرقام بالحروف العربية.
الوصف بالإنجليزية: Tafqit is an open-source JavaScript library designed to convert numbers into Arabic text using the "Tafqit" system. It supports both integer and decimal numbers, converting them into Arabic words, such as turning "123" into "مئة وثلاثة وعشرون". This library is useful for applications like invoices, financial documents, or any app requiring number-to-text conversion in Arabic.
