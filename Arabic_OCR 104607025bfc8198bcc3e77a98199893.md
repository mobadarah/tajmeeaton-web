# Arabic_OCR

الوصف بالعربية: The Arabic NLP tools is in disarray. Most resources require affiliation with a university and are scattered across the web. This is an attempt to rectify that. This repository will attempt to provide Python wrappers (and perhaps eventually brew formulae!) for many of our field's tools.
