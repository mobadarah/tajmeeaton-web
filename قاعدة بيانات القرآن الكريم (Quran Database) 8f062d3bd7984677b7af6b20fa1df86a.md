# قاعدة بيانات القرآن الكريم (Quran Database)

التقنيات: Database, SQLite
وسوم: القرآن الكريم
نوع المبادرة: قاعدة بيانات, مصادر
الوصف بالعربية: يوفر هذا المشروع قاعدة بيانات شاملة للقرآن الكريم بصيغة SQLite، تحتوي على معلومات حول السور، بما في ذلك أسماؤها بالعربية، نطقها بالإنجليزية، تصنيفها (مكية أو مدنية)، عدد الآيات، ومحتوى السور.
الوصف بالإنجليزية: This project offers a comprehensive SQLite database of the Holy Quran, including information about Surahs such as their Arabic names, English pronunciations, classifications (Meccan or Medinan), number of verses, and content.
