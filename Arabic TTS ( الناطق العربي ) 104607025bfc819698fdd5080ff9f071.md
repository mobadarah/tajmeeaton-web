الوصف بالعربية: HTS Arabic voice for Festival TTS: is an arabic voice trained by HTS technologie to work on Festival TTS system. The voice is HMM trained on Nawar Arabic Speech Corpus. You can test the voice with diacritics on: http://ar.arabicspeechcorpus.com/ Install necessary packages: sudo apt-get install festival sudo apt-get install htsengine You can install Festival from pakcage manager from your linux distribution.
