# salah

التقنيات: Rust
وسوم: مواقيت الصلاة 🕋
نوع المبادرة: مكتبة برمجية
الوصف بالعربية: مكتبة مفتوحة المصدر مكتوبة بلغة Rust لحساب مواقيت الصلاة الإسلامية. توفر دقة في حساب أوقات الصلاة لجميع المواقع الجغرافية، مع مراعاة اختلافات المذاهب الإسلامية.
الوصف بالإنجليزية: An open-source library written in Rust for calculating Islamic prayer times. It offers accurate prayer times for any geographical location, considering the differences among Islamic schools of thought.
