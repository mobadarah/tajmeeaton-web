# قاعدة بيانات للقرآن الكريم SQLite

التقنيات: Database, Dataset, SQLite
وسوم: إسلام, القرآن الكريم, تعليم, تقنية, قاعدة بيانات
نوع المبادرة: قاعدة بيانات
الوصف بالعربية: قاعدة بيانات للقرآن الكريم فيها السور والآيات وبيانات أخرى مثل نوع السورة مدنية أم مكية ونوع الآية وعدد آيات السورة وغيرها من البيانات.
الوصف بالإنجليزية: SQLite database for the Holy Quran. Containing tables for the chapters and the ayat (verses). It contains information like the type of the verse and chapter, the count of verses in a chapter and more other data.
