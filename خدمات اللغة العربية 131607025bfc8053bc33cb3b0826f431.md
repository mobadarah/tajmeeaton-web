# خدمات اللغة العربية

التقنيات: Java, JavaScript, TypeScript
وسوم: التشكيل, اللغة عربية, معالجة اللغة الطبيعية
نوع المبادرة: تطبيق ويب, مكتبة برمجية
الوصف بالعربية: خدمات اللغة العربية هي مكتبة برمجية متعددة الاستخدامات تقدم وظائف مساعدة لمعالجة وتحويل النصوص العربية. تشمل الميزات إزالة التشكيل، إزالة التطويل، تحويل النص إلى الخط العربي القديم، واستبدال الكلمات المحظورة بحروف مشابهة بصريًا.
الوصف بالإنجليزية: Arabic Services is a versatile library offering utility functions for processing and transforming Arabic text. Features include removing diacritics, removing elongation characters, converting text to old Arabic script, and replacing banned words with visually similar characters.
