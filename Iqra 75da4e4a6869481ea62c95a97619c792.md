# Iqra

التقنيات: CSS, Express.JS, Flask, HTML, Java, Node.JS, Python, ReactJS, Speech recognition
وسوم: القرآن الكريم
نوع المبادرة: RESTful API, تطبيق iOS, تطبيق أندرويد, تطبيق ويب
الوصف بالعربية: منصة مفتوحة المصدر تمكّن المستخدمين من البحث في القرآن الكريم باستخدام تقنيات التعرف على الصوت عبر تطبيقات مختلفة للأجهزة المحمولة والويب. تتيح المنصة للمسلمين الوصول إلى النصوص القرآنية بسهولة عبر تكنولوجيا البحث الصوتي.
الوصف بالإنجليزية: Iqra is an open-source platform that allows users to search the Quran using speech recognition technology. The platform provides seamless access to Quranic texts through mobile and web applications.
