# التجميعة

دليل المشاريع العربية مفتوحة المصدر. منصة curated تضم أكثر من ١٠٠ مشروع تقني عربي مفتوح المصدر، مع إمكانية البحث والتصفية حسب التقنيات والرخص والحالة.

## المميزات

- دليل شامل للمشاريع العربية مفتوحة المصدر
- بحث سريع وتصفية متقدمة
- دعم الوضع المظلم
- تصميم متجاوب مع جميع الأجهزة

## التقنيات المستخدمة

- [Astro](https://astro.build/) — إطار عمل لبناء المواقع الثابتة
- خط [Noto Nastaliq Urdu](https://fonts.google.com/noto/specimen/Noto+Nastaliq+Urdu) للعناوين
- خط [ثمانية](https://font.thmanyah.com/) للمحتوى

## البدء محلياً

```bash
cd site
npm install
npm run dev
```

## البناء والنشر

```bash
cd site
npm run build
```

الناتج يكون في مجلد `site/dist`.

## المساهمة

نرحب باقتراحاتكم وإضافاتكم. تفضلوا بفتح issue أو pull request.

## الرخصة

هذا المشروع مرخص تحت رخصة وقف من أعجوبة.
