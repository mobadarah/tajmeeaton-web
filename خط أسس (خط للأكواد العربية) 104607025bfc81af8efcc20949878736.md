# خط أسس (خط للأكواد العربية)

وسوم: اللغة عربية, تقنية, خط, فونت, ويب
نوع المبادرة: خط عربي
الوصف بالعربية: خط أسس أحادي المسافة، صمم بشكل رئيسي لمحررات أكواد لغة الأسس وهو متاح الآن لستخدمه كيفما تشاء ومع أي لغة أحببت.
الوصف بالإنجليزية: Monospace font for Arabic code that may contain Latin-like text. It was built mainly for the code editor of Alusus programming language but can be used as other use cases.
