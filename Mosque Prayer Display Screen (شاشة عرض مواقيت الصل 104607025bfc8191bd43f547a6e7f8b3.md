# Mosque Prayer Display Screen (شاشة عرض مواقيت الصلاة للمسجد)

التقنيات: Next.JS, Tailwind CSS, TypeScript
وسوم: مواقيت الصلاة 🕋
نوع المبادرة: تطبيق ويب
الوصف بالعربية: تطبيق مفتوح المصدر يتيح للمساجد عرض مواقيت الصلاة على شاشات للمصلين، ويعمل أيضًا كتطبيق ويب تقدمي يمكن استخدامه دون اتصال بالإنترنت على أي متصفح حديث.
الوصف بالإنجليزية: An open-source application that allows mosques to display prayer times on screens for worshipers. It also functions as an offline progressive web app compatible with any modern web browser.
