# Klaam

التقنيات: Jupyter Notebook, Machine learning, Python, Speech recognition
وسوم: مكتبة
نوع المبادرة: CLI, مكتبة برمجية
الوصف بالعربية: نظام تعرف على الكلام العربي، تصنيفه، وتحويل النص إلى كلام باستخدام نماذج متقدمة مثل Wave2Vec وFastSpeech2. يوفر ادوات للتدريب والتنبؤ باستخدام نماذج مدربة مسبقاً لتحسين الأداء في مهام التعرف على الكلام وتحويل النص إلى صوت.

الوصف بالإنجليزية: Arabic speech recognition, classification and text-to-speech using many 
advanced models like wave2vec and fastspeech2. This repository allows 
training and prediction using pretrained models.
