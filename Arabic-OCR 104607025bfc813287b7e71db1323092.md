# Arabic-OCR

الوصف بالعربية: Model uses Named Entity Recognition concept to tag words in arabic text. It consists of Bi-directional GRU units(One forward and the other is backward) and a CRF layer referenced from https://arxiv.org/pdf/1508.01991v1.pdf Model is trained on ANERCorp dataset. And uses FastText's Arabic vectors for word embedding. No.
