# PyArabic

التقنيات: Python
وسوم: أنواع الكلمات, التشكيل, اللغة عربية, تحليل صرفي, تقنية
نوع المبادرة: مكتبة برمجية
الوصف بالعربية: مكتبة برمجية للغة العربية بلغة بيثون، توفر دوالّ للتحكم في الحروف والنصوص، مثلا تحديد نوع الحرف، حذف الحركات، مقارنة التشكيل.
الوصف بالإنجليزية: A specific Arabic language library for Python,
 provides basic functions to manipulate Arabic letters and text, like 
detecting Arabic letters, Arabic letters groups and characteristics, 
remove diacritics etc.
