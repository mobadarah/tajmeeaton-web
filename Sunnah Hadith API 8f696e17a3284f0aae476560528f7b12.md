# Sunnah Hadith API

التقنيات: API, Express.JS, Hadith, JSON, JavaScript, Node.JS
وسوم: الحديث النبوي
نوع المبادرة: RESTful API, سكريبت برمجي
الوصف بالعربية: هذا API بسيطة يقوم بسحب البيانات من موقع http://sunnah.com/
ميزته أنه يوفر البحث عن الاحاديث سواء بنص عربي أو انجليزي والنتائج تكون مترجمة باللغة العربية والإنجليزية


الوصف بالإنجليزية: This is a simple API that scrapping data from the sunnah.com website. This API provide translation of the arabic hadith in English (Mainly English)
