# tarteel-ml

التقنيات: Artificial intelligence, Dataset, Jupyter Notebook, Machine learning, Python, Speech recognition
وسوم: القرآن الكريم
نوع المبادرة: سكريبت برمجي
الوصف بالعربية: مشروع مفتوح المصدر يستخدم تقنيات تعلم الآلة للتعرف على تلاوة القرآن الكريم من خلال الصوت. يهدف المشروع لدعم تحليل وتحسين تلاوة القرآن باستخدام الذكاء الاصطناعي وتطوير أدوات للتعرف على الآيات والأخطاء.
الوصف بالإنجليزية: An open-source project utilizing machine learning to recognize Quranic recitation through voice. The project aims to support the analysis and improvement of Quran recitation using AI, providing tools for verse recognition and error detection.
