# Wojood - Nested Arabic NER

التقنيات: Dataset, Machine learning, NLP, Python
وسوم: اللغة عربية, محررات نصوص
نوع المبادرة: Model, مكتبة برمجية, ورقة بحثية
الوصف بالعربية: Wojood عبارة عن مجموعة من النصوص للتعرف على الكيانات المسماة المتداخلة (NER) باللغة العربية. تحدث الكيانات المتداخلة عندما يتم تضمين ذكر كيان داخل ذكر كيان آخر.
الوصف بالإنجليزية: Wojood is a corpus for Arabic nested Named Entity Recognition (NER). Nested entities occur when one entity mention is embedded inside another entity mention.
