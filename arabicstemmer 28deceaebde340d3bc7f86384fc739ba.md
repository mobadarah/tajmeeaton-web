# arabicstemmer

التقنيات: Makefile, Python, Snowball
وسوم: اللغة عربية, ايجاد جزور الكلمات, معالجة اللغة الطبيعية, ويب
نوع المبادرة: CLI, تطبيق ويب, مكتبة برمجية
الوصف بالعربية: هذه خوارزمية لكتابة النصوص العربية مكتوبة بلغة إطار عمل Snowball. وهي توفر كتابة نصية بسيطة وتطبيع النص.
الوصف بالإنجليزية: An algorithm for Arabic stemming written on Snowball framework language. If offers light stemming and text normalization.
